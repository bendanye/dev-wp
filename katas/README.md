# Katas

## Setup

1. Install jq
2. Create a new file, `schedule.json` that contains the following:

```
{
    "katas": [
        {
            "name": "<<name of the kata>>",
            "day-of-week": "<<day of week. For example Monday>>",
            "contains-multiple-kata": <<true|false>>,
            "repo_dir": "<<the location of the kata>>"
        }
    ]
}
```
